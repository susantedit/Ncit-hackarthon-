## About

Sathi AI is an intelligent multilingual AI platform created to make trustworthy information more accessible for people in Nepal. It combines conversational AI, government service guidance, market intelligence, AI-powered price verification, and fact checking into one seamless experience.

## Features

| Module | Description |
|--------|-------------|
| 🤖 AI Chat | Conversational AI assistant in Nepali and English |
| ✅ Fact Checker / Realtime Verify | Autonomous claim extraction, evidence retrieval, contradiction scoring, and verdicts |
| 🎬 Creator Mode | Generate scripts and voice for reels, YouTube, and podcasts |
| 📚 Study Mode | Explain, simplify, revise, or quiz any topic across 3 depth levels |
| 🧘 Focus Mode | Custom Pomodoro timer with guided breathing and completion overlay |
| 📅 Productivity Planner | Set a goal and get a voice-guided daily plan |
| 🛡️ Safety Guardian | SOS hold button, fake call, live GPS and email alerts, weather widget |
| 🌍 Voice Translator | Translate and hear text in 80+ languages including Nepali |
| 🎙️ Podcast Studio | Turn URLs, PDFs, YouTube links, or prompts into multi-voice podcasts with RAG chat |
| 📖 Voice Journal | Speak your daily entry; AI reflects it back |
| 🎮 Brain Games | Interactive games — quiz, debate, riddles, mood, and more |
| 🎭 CageBait | Scam-honeypot personas with intel extraction and session summaries |
| 🎓 Interviews | Create, run, and review mock interviews with coaching feedback |
| 📋 History | All sessions saved per user in MongoDB |
